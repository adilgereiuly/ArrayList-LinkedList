# ArrayList-LinkedList

## Project Description
This project implements custom data structures from scratch without using Java's built-in collections framework.

## Implemented Classes

| Class | Description |
|-------|-------------|
| `MyList<T>` | Generic interface defining common list operations |
| `MyArrayList<T>` | Dynamic array implementation that grows automatically |
| `MyLinkedList<T>` | Doubly linked list implementation with head and tail pointers |
| `MyStack<T>` | LIFO stack using MyArrayList as underlying storage |
| `MyQueue<T>` | FIFO queue using MyArrayList as underlying storage |
| `MyMinHeap<T>` | Min-heap using MyArrayList, smallest element always at root |

## How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the `Main` class to execute all test cases

## Test Output

The `Main` class tests all methods including:
- Adding/inserting elements
- Removing by index and by value
- Stack push/pop/peek operations
- Queue enqueue/dequeue/front operations
- MinHeap add/peek/removeMin operations
